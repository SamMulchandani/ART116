Why I want to learn about web design:

I want to learn web design because it is a useful skill in today’s world. Almost every business or organization needs a website, so understanding how websites are built and designed can be helpful in many different fields. I think learning the basics of web design will give me a better understanding of how the internet works and how people interact with online content.
Additionally, I was lacking an art class on my transcript, and it is a requirement for several universities I am currently applying to, thus, a web design class at my local community college was my best bet to fulfill that requirement.
I’m also interested in learning how to make a website look clean and organized. Good web design can make information easier to find and understand, which is important for anyone who wants to share ideas online. Even if I don’t plan to become a web designer, I think having some knowledge of HTML, CSS, and general design principles could be useful for future projects or jobs. Overall, I want to learn web design because it’s a practical skill that can be applied in many different situations.
